# Healthcare App - Angular Frontend

Patient management system with authentication and caregiver assignment.

## Features

- JWT Authentication
- Patient CRUD operations
- Multiple caregiver assignment
- Role-based access control
- Responsive design

## Setup

```bash
# Install dependencies
npm install

# Start dev server
npm start
```

The app runs at `http://localhost:4200`

## Default Login Credentials

- **Admin**: admin / Password123!
- **Manager**: manager / Password123!
- **User**: user1 / Password123!

## Tech Stack

- Angular 17
- TypeScript
- RxJS
- HttpClient
- Reactive Forms
